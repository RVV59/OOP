# Проект "Интернет-магазин по продаже бытовой техники"

В проекте представлены классы `Product` и `Category`, которые реализуют базовые возможности для работы с продуктами и категориями. `Product` описывает основные характеристики продукта, такие как название, описание, цена и количество в наличии. `Category` объединяет продукты одной категории и предоставляет функционал для учета количества продуктов и категорий.

## Описание классов

#### Product

- **Название (name)**: Полное название продукта.
- **Описание (description)**: Подробное описание продукта.
- **Цена (price)**: Цена продукта.
- **Количество в наличии (quantity)**: Текущее количество продукта на складе.

#### Category

- **Название (name)**: Название категории.
- **Описание (description)**: Описание категории.
- **Список товаров категории (products)**: Список продуктов, относящихся к данной категории.
- **Количество категорий (category_count)**: Количество категорий.
- **Количество товаров (product_count)**: Количество продуктов.
